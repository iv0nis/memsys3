# GitHub - Pujar canvis al repositori

## Instruccions

- Puja el repo a GitHub en català, sense emojis ni signatures
- Commits atòmics i descriptius
- Afegeix el tag corresponent si escau
- No pugis la versió més enllà de la 0.x.x fins que sigui estable
- Només augmenta el minor si hi ha canvi rellevant, en cas contrari augmenta el patch

## Workflow

1. Revisar canvis: `git status` i `git diff`
2. Afegir arxius: `git add .` o selectiu
3. Commit amb missatge descriptiu
4. Push a origin: `git push origin master`
5. Si escau, crear tag: `git tag v0.x.x && git push --tags`
