# memsys3/ (dogfooding)

Esta carpeta es la **instancia de dogfooding** de memsys3 — el propio sistema usándose a sí mismo para desarrollarse.

- **NO se distribuye**. Lo que se distribuye vive en `memsys3_templates/`.
- Para la propuesta de valor, instalación y workflow del sistema → ver [`../README.md`](../README.md) (raíz del repo).
- Para la doc del subsistema de memoria → ver [`memory/README.md`](memory/README.md).

Ver también ADR-011 y ADR-025 (estrategia dogfooding vs distribuible).
