# agent-identity.md

Se te ha asignado un agent_id para esta sesión: **Agent [X]**.

Guarda tu identidad en `~/.claude/memsys3_agent_id` si no está ya:

```bash
echo "Agent X" > ~/.claude/memsys3_agent_id
```

> En Windows nativo usa `%USERPROFILE%\.claude\memsys3_agent_id`.

Usa este ID para identificarte en reuniones, documentación y comunicación con el moderador.
